# Heart Rate Detection from Video using Eulerian Magnification

## Overview
This project estimates human heart rate from facial video using computer vision and signal processing techniques.

## Methodology
- Video preprocessing using OpenCV
- Region of Interest (ROI) extraction from facial area
- Laplacian Pyramid construction for multi-scale decomposition
- Eulerian Video Magnification to amplify subtle signals
- FFT-based temporal bandpass filtering (1–1.8 Hz)
- Peak detection to estimate heart rate (BPM)

## Technologies Used
- Python
- OpenCV
- NumPy
- SciPy

## Applications
- Remote health monitoring
- Contactless vital sign measurement
- Telemedicine systems
